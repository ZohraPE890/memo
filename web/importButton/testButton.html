Configuration du bouton « Importer dans MEMO»

Contexte

Le bouton MEMO va permettre aux utilisateurs de votre site d'importer les offres qu'ils choisissent sur votre site dans leur tableau de bord MEMO qui regroupe l'ensemble de leurs candidatures (offres issues des job boards, candidatures spontanées, réseau personnel…).

Vous pouvez voir des exemples d'intégration sur les sites de La Bonne Boîte et de Pôle emploi.

MEMO est un produit opéré par Pôle emploi.

Etape 1 : Positionner une ou plusieurs balises <a>

Placez une balise <a> à l'emplacement où vous voulez voir apparaître un bouton d'import MEMO.

<a class="memo-import-button" data-url="[url_de_l_offre_a_importer]" data-style="[classe css]" data-title="[titre de l'offre]"></a>

class : mettre la valeur de classe memo-import-button pour que cette balise <a> soit identifiée et qu'un bouton MEMO soit insérée dedans

data-url : L'attribut data-url doit contenir l'adresse de l'offre à importer. Il doit être encodé avec la méthode javascript encodeURIComponent ou équivalent.

data-style : (facultatif) une classe à appliquer au bouton. LISTE DES CLASSES A DEFINIR

data-title : (facultatif) Le titre de l'offre.

Etape 2 : Script de génération des boutons

Placez le snippet javascript suivant avant la balise </body> afin de convertir les balises <a> en boutons d'import. Pour information, ce script réalise un traitement asynchrone qui ne ralentira pas le chargement des autres ressources de la page.

<script>
!function(d,s,id)
{
var js, fjs= d.getElementsByTagName(s)[0];
if(!d.getElementById(id)) {
js= d.createElement(s);
js.id=id;
js.async=true;

js.src="https://memo.pole-emploi.fr/js/importButton/memoButton-min.js";
fjs.parentNode.insertBefore(js,fjs);
}
}(document,"script","memoScript");
</script>

Setting up the « Import into MEMO » button

Context

The MEMO button allow your site's visitors to import job offers in their MEMO dashboard. This dashboard contains all their candidacies (job positions found on job boards, spontaneous applications, personal network opportunities…).

You can see examples of "import" buttons on La Bonne Boîte and Pôle emploi

MEMO is operated by Pôle emploi.

Step 1 : Position one or several <a> tags

Put an <a> tag where you want the "import" button to appear.

<a class="memo-import-button" data-url="[url_of_the_job_offer_to_import]" data-title="[offre_title]" data-style="[button_]"></a>

class : put the value memo-import-button for this <a> tag to be transformed in a MEMO button.

data-url : This attribute must contain the job offer address and be URLEncoded. (use encodeURIComponent in javascript).

data-style : (optional) a css class to apply to the button.

data-title : (optional) the offer title.

Step 2 : Button generator script

Put this javascript snippet before the </body> tag in order to transform the <a> tags into "import" buttons. Note that this script is asynchronous and will not slow the loading of the rest of the page.

<script>
!function(d,s,id)
{
var js, fjs= d.getElementsByTagName(s)[0];
if(!d.getElementById(id)) {
js= d.createElement(s);
js.id=id;
js.async=true;

js.src="https://memo.pole-emploi.fr/js/importButton/memoButtonPE-min.js";
fjs.parentNode.insertBefore(js,fjs);
}
}(document,"script","memoScript");
</script>
